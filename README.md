# bishehuatu
毕设论文智能AI画图助手，是一款面向毕业设计、课程设计和论文配图场景的AI工具。用户可以上传系统说明、SQL文件、README文档、项目代码或需求描述，让AI自动分析系统业务数据库结构功能模块和代码逻辑，快速生成论文中常用的系统架构图功能模块图、ER图、流程图、类图、用例图、时序图以及三线表，适合不会画图、论文图表不规范、代码结构看不懂、答辩前需要快速整理系统设计材料的同学使用。相比手动画图，AI可以自动识别模块关系、表关系、业务流程和类结构，减少重复排版和修改时间，让论文配图更完整、更统一，也更适合放进开题报告、论文正文和答辩PPT。  除了自动生成图表，小程序还支持AI讲解代码、分析项目结构、解释接口逻辑，并可辅助修改代码。无论是想补论文里的系统设计图，还是想快速理解自己的毕业设计项目。
